Инструкция

  1. Выберите приложение
  2. Нажмите на него
  3. Затем нажмите "разрешить"
  4. Скопируйте часть адресной строки от access_token= до &expires_in

Инструкция для приложений с авторизацией

  1. Выберите приложение
  2. Нажмите на него
  3. Затем в ссылке укажите после username= - свой телефон, после password= - пароль (спец. символы в URL кодировку)
  4. Если включена двухэтапная авторизация, то после нажатия Enter добавьте code= к ссылке с полученным кодом
